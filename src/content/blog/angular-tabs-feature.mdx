## API Reference

### Tabs Component

The `Tabs` component is the wrapper that manages tab state and renders the tab navigation.

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `defaultValue` | `string` | - | The value of the tab that should be active by default |
| `values` | `Array<{label: string, value: string}>` | - | Array of tab definitions (alternative to using Tab components) |

#### Usage Patterns

**Pattern 1: ContentChildren (Recommended)**
```html
<Tabs>
  <Tab value="tab1" label="Tab 1" default>Content 1</Tab>
  <Tab value="tab2" label="Tab 2">Content 2</Tab>
</Tabs>
```

**Pattern 2: Values Array**
```html
<Tabs values={[
  {label: 'Tab 1', value: 'tab1'},
  {label: 'Tab 2', value: 'tab2'}
]}>
  <Tab value="tab1">Content 1</Tab>
  <Tab value="tab2">Content 2</Tab>
</Tabs>
```

### Tab Component

The `Tab` component represents individual tab content.

#### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | **Required** | Unique identifier for the tab |
| `label` | `string` | **Required** | Display label for the tab button |
| `default` | `boolean` | `false` | Whether this tab should be active by default |

```typescript
      </div>

})
export class TabsComponent {
  @ContentChildren(TabComponent) tabs!: QueryList<TabComponent>;
  // ... implementation
}
```

```typescript
// Tab Component (tab.component.ts)  
@Component({
  selector: 'Tab',
  template: \`
    <div *ngIf="active">
      <ng-content></ng-content>
    </div>
  \`
})
export class TabComponent {
  @Input() value!: string;
  @Input() label!: string;
  @Input() default: boolean = false;
  active: boolean = false;
}
```

## Best Practices

### 1. Use Semantic Values
```html
<!-- Good -->
<Tab value="installation" label="Installation">

<!-- Avoid -->
<Tab value="tab1" label="Tab 1">
```

### 2. Provide Meaningful Labels
```html
<!-- Good -->
<Tab value="npm" label="npm">
<Tab value="yarn" label="Yarn">

<!-- Avoid -->
<Tab value="npm" label="Option 1">
```

### 3. Mark Default Tabs Clearly
```html
<Tabs>
  <Tab value="overview" label="Overview" default>
  <Tab value="api" label="API Reference">
  <Tab value="examples" label="Examples">
</Tabs>
```
